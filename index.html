Tecnologías Utilizadas

🎨 Frontend

  • HTML5: Estructura semántica con header, nav, main, sections
  • CSS3: Variables CSS, Flexbox, Grid, Animations, Media Queries
  • JavaScript ES6+: Async/await, Arrow functions, Template literals
  • Responsive Design: Mobile First, optimizado para dispositivos móviles

🗄️ Base de Datos y Almacenamiento

  • Base de Datos: Almacenamiento centralizado de datos CUIT y consultas
  • localStorage: Para sesión de usuario y notas
  • JSON estructurado: Para datos de CUIT (consulta_masiva.json)

🏛️ APIs e Integraciones

  • API ARCA (ex AFIP): Obtención y validación de datos tributarios de contribuyentes
  • API BCRA: Central de Deudores del Sistema Financiero (Fase 2)
  • API BCRA: Consulta de situación crediticia de personas y empresas (Fase 2)
  • Base de Datos Riesgo: Almacenamiento de resultados y histórico de consultas

🛠️ Herramientas de Desarrollo

  • Visual Studio Code: IDE principal
  • Chrome DevTools: Testing y debugging
  • Git: Control de versiones
