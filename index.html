Objetivos del Proyecto

Consulta de CUITs

Consultas tributarias individuales y masivas con datos de ARCA (ex AFIP)

Riesgo Crediticio

Evaluacion de riesgo con BCRA, alertas tempranas y score de confianza

Precios Combustibles

Obtencion automatizada de precios desde Axion y Shell en tiempo real

Seguridad

Autenticacion segura con usuarios, sesiones y control de acceso

Mobile First

Diseño responsive para smartphones, tablets y desktop

Base de Datos

Almacenamiento centralizado de consultas, precios e historiales
